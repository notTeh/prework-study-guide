CSS

  • CSS: A programing language used by developers to define how webpage content is presented to users.
  • CSS defines colors, aesthetics, visual layout, and organzation.
  • Inline CSS: Apply to a specific HTML tag - Uses style attributes with a CSS role to style a particular page element only.
  • Internal Style Sheet: Contains CSS roles for the webpage in the head section of the HTML.
  • Margin: Indicates how much space we want around the OUTSIDE of an element.
  • Padding: Adds space around the content INSIDE an element.
  • DRY CODE: Do not repeat yourself - A coding principle that encourages developers to reuse or share pieces of a code to reduce number of lines of code that need to be written or maintained.
