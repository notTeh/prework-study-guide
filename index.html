HTML

  • HTML Attribute: Defines additional information about an element.
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • Semantic HTML Elements: Name of element describes its content and role on the site.
  • Metadata: Data about our webpage that the browser uses to effectively serve our wepage to users.
  • Child Elements: Elements that are nested inside other elements.
  • When working with visible HTML elements, the content is contained between an opening and closing tag.

CSS

  • CSS: A programing language used by developers to define how webpage content is presented to users.
  • CSS defines colors, aesthetics, visual layout, and organzation.
  • Inline CSS: Apply to a specific HTML tag - Uses style attributes with a CSS role to style a particular page element only.
  • Internal Style Sheet: Contains CSS roles for the webpage in the head section of the HTML.
  • Margin: Indicates how much space we want around the OUTSIDE of an element.
  • Padding: Adds space around the content INSIDE an element.
  • DRY CODE: Do not repeat yourself - A coding principle that encourages developers to reuse or share pieces of a code to reduce number of lines of code that need to be written or maintained.

Git

  • GIT: Version control system that lets you manage and keep track of versions of your project.
  • git: must precede every git command
  • checkout: move working branch to new branch
  • -b: creates new branch
  • status: checks what branch you're currently in
  • origin: indicates source of pull will be in github repo
  • Header: h1-h6, h1 being the biggest

JavaScript

  • Javascript: Lightweight language that provides logic and interactivity for websites.
  • NULL: Null type is inhabited by exactly one value - Null; Indicates the absence of an object.
  • Undefined: Undefined type is inhabited by exactly one value - undefined; indicates the absence of a value.
  • Boolean: Boolean type represents a logical entity and is inhabited by two values: true and false.
  • Number: Number type is a double-precision 64-bit binary format IEEE 754 value. Capable of storing positive flating-point numbers between 2^-1074 and 2^1023.
  • BigInt: BigInt type is a numeric primate in JavaScript that can represent integers with arbitrary magnitude.
  • String: String type represents textual data and is encoded as a sequence of 16-bit unsigned integer values representing UTF-16 code units. All strings are contained in "Quotation Marks" and end with a semicolon.
  • Symbol: Symbol type is a unique and immutable primate value and may be used as the key of an object property. This can be Anything. Everything in JavaScript is an object and can be stored in a variable.
  • Objects: In computer science, an object is a value in memory which is possibly reference by an identifier.
  • Array: Structure that allows you to store multiple values in a single reference.
  • Variable: A named container that allows us to store data in our code.
  • Control Flow: The order in which a computer executes code in a script.
  • Loops: A control Flow statement that allows you to execute a block of code repeatedly as long as a specified condition is true.
